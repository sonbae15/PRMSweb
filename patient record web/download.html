DOWNLOAD APPLICATION HERE!


The Android application for the system can be downloaded here on the Patient Records Management System website. The application's primary purpose is to set reminders for the patients regarding their medication in a form of alarm. The users will be asked for their basic information for registration. Please make sure that the RFID's serial number to be inputted during the registration is correct and valid. The assigned physician can post announcements in the application for the users so please make sure to register the correct RFID number.
